# Mega Tetris 🧩

**200×100 巨型俄罗斯方块** — 大画布、自定义配色、AI 自动玩、背景音乐

## 功能

| 特性 | 说明 |
|------|------|
| 🖥️ **超大棋盘** | 200列 × 100行，方块 10px，占满整个浏览器窗口 |
| 🎨 **自定义配色** | 左侧面板可独立调节背景色、网格线、7种方块颜色、幽灵颜色及透明度 |
| 🤖 **AI 自动玩** | 一键开启 AI 自动游戏，可选 **软降**（一格一格下落）或 **直落**（瞬间到底） |
| 📦 **积累消除** | 可设置积累 N 行再一起消，堆叠越多消除越壮观 |
| 🎵 **背景音乐** | 2 首电音曲目可选，支持开关、切换、音量调节 |
| 💥 **消行特效** | 彩色粒子爆炸动画 |
| 👆 **触摸支持** | 移动端滑动控制：左右滑动移动、上下滑动加速、轻点旋转 |
| ⌨️ **键盘控制** | `←→` 移动 / `↑` 旋转 / `↓` 加速 / `空格` 直落 / `P` 暂停 |

## 快速开始

直接打开 `index.html` 即可游玩，无需任何依赖。

```bash
# 或者用本地服务器
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 配色预览

默认多巴胺配色方案：

```
I — 金橙    #f59f00    O — 石青蓝   #4dabf7
T — 电光紫  #9775fa    S — 霓虹绿松 #20c997
Z — 珊瑚粉  #ff6b6b    J — 霓虹蓝   #339af0
L — 紫罗兰  #cc5de8
```

左侧面板可实时调整所有颜色。

## 操作

| 按键 | 功能 |
|------|------|
| `←` `→` | 左右移动 |
| `↑` | 旋转 |
| `↓` | 加速下落 |
| `空格` | 直落到底 |
| `P` | 暂停/继续 |
| `双击` | 暂停/继续 |
